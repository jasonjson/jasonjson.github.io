<p>**<em>Given an array of size n, find the majority element. The majority element is the element that appears more than $\floor</em> {n/2}$ times***<br />
[expand title="code"]</p>
<pre>
public class Solution {
    public int majorityElement(int[] nums) {
        int key = -1, count = 0;
        for (int n : nums) {
            if (count == 0) {
                key = n;
                count ++;
            } else {
                if (key == n) {
                    count ++;
                } else {
                    count --;
                }
            }
        }
        return key;
        //bug: return count
    }
}
</pre>
